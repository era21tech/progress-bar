# Mini Circular Progress

A small JavaScript project that shows an animated circular progress bar with a numeric percentage inside. Built using HTML, CSS (conic/stroke animation) and a tiny JS interval counter.

## Preview
- Numeric percentage in the center.
- Gradient-colored SVG ring that animates to the target percent.
- Clean, minimal UI using shadows and rounded shapes.

## Files
- `index.html` ‚Äî markup, SVG ring and the JS script. 
- `style.css` ‚Äî all styles and the SVG stroke animation.

## How to run
1. Clone or download the project folder.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
